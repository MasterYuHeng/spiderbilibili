# 社区行为准则

## 我们希望的协作方式

为了让 SpiderBilibili 的开源协作保持高效和可持续，所有参与者都应尽量做到：

- 尊重不同背景、经验和观点的贡献者。
- 以事实、代码和可复现信息为基础讨论问题。
- 在提出质疑时聚焦方案与实现，而不是针对个人。
- 接受建设性反馈，并愿意根据证据修正结论。

## 不被接受的行为

以下行为不适合出现在项目协作中：

- 人身攻击、羞辱、骚扰或威胁
- 恶意挑衅、反复刷屏、故意制造对立
- 泄露他人的隐私、凭据或未公开安全信息
- 明知会破坏讨论或协作秩序的行为

## 执行原则

- 维护者可以删除明显不适当的内容。
- 对持续破坏协作环境的账号，维护者可以限制其讨论或贡献权限。
- 处理过程会尽量以维护项目正常协作为目标，并保持必要的克制与透明。

## 适用范围

本准则适用于仓库 Issue、Pull Request、讨论区以及与本项目直接相关的协作沟通场景。
